# Example Web Scraping Project for Home Advantage in Test Cricket

This project demonstrates how web scraping can be used to collect and analyse publicly available cricket match data in order to study **home advantage in Test cricket**.

The project focuses on extracting match-related information from websites, structuring the data, and preparing it for analytical exploration of performance differences between home and away teams.

---

## Project Overview

The purpose of this project is to investigate whether teams perform better when playing **Test matches at home** compared to away conditions.

In this project, I:
- Scraped data from web pages using Python
- Parsed and extracted relevant information from HTML content
- Cleaned and structured the data into a usable format
- Prepared the dataset for downstream analysis or reporting
